Triggers

For game panel

  • hover ON
    • 400ms delay
  • Hover OFF
  • Onclick

Rules

For game panel

  • hover ON
    • Cursor switches from arrow to finger icon
    • After 400ms Delay
      • Pop-up appears with game description
  • Hover OFF
    • Cursor reverts back to arrow icon
    • Pop-up closes
  • Onclick
    • Navigates to game store page
